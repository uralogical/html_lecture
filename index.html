div で遊ぶ

  • 名前をつける
  • 色を変える
  • 大きさを変える

ひとまず 🟥 をつくる
✍️✍️✍️

📛名前をつける

📛名前をつける

  • class
    • 👉 汎用的な名前をつけるときに使う
    • . を使って指定する
  • id
    • 👉 一意の名前を付けるときに使う
    • # を使って指定する

🎨色を変える

🎨色を変える

  • background-color を使う
  • 代表的な名前での指定
    • 👉 blue, yellow など
  • カラーコードでの指定
    • 👉 #333, #00A4AC など

🔎大きさを変える

🔎大きさを変える

  • width, height を使う
    • 👉 px, %, vh などで指定可能
やってみよ👏

🔴形を変える

🔴形を変える

  • 🟥 👉👉👉 🔴
  • border-radius を使う
  • px, % での指定
やってみよ👏

🚌配置を変える

🚌配置を変える

  • position を使う
  • relative, absoluteでの指定
  • display: flex を使う
    • justify-content: center;
    • align-items: center;
